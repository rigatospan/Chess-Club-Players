Chess Teams Manager

Local software to create/manage/update chess teams and players info of a local club automatically. It is comprised by 7 total sections:

  1. Configure Players List (top-left): define the critiria for searching the players database

  2. Create New Team (bottom-left): crete teams and set team's inforation

  3. Players Database (top-middle): the club's players database to be searched-restricted

  4. Created Teams Frames (bottom-middle): the teams compositions to be filled with players

  5. Configure Database Columns (top-right): configure the players database columns with additional inforation

  6. Move Players to Teams (middle-right): select team to place the selected players

  7. Update Database (bottom-right): create/update the players database

1. Configure Players List

On this panel you can define the searching critiria in the players database. There are 7 total critiria to impose:

  1. elo type and range: choose one of the elo type and its minimum and maximum rate. When you click apply only the players in the specified elo-type range will be displayed.

  2. age: choose the age range of the players to be displayed.

  3. gender: choose the gender of the players, 'All', 'M' for males, 'F' for females.

  4. active players: when selected yes only the fide active players will be displayed, when no only the inactive.

  5. Eso fee paid: when selected yes only the players with paid eso fee will be displayed.

  6. Search by name: search a player/players by name, either with latin or greek characters. When some text is insert in the box, only the players with names that start from that text will be displayd when apply.

There are two buttons on this section as well:

  1. Apply: when clicked will display the players based on the current insertions on the above fields. Of course all fields are combined to restrict the database, i.e. one can search all female, active players, with fide elo in range 1500-2000, aged between 10-20.

  2. Clear: when clicked it clears all entries to the fields and automatically reset the database to its initial display of all players.

2. Create New Team

On this panel you can create multiple teams to be fill with players from the database. There are 8 field to insert information about the match to take place:

  1. Team's Name: provide the name of the club's team, i.e. So Aigaleo Under 10 A

  2. Opponent Team's Name: provide the name of the opponent club's team, i.e. So Peristeriou Under 10 B

  3. Tournament: provide the name of the competition that the team takes part, i.e. Eso tournament Under 10 2023

  4. Match Date: probide the date of the match to take place: i.e. 1/1/2030

  5. Address: provide the address of the place when the match is hold

  6. Round: provide the number of the round, i.e. 7th round

  7. Home Court: if selected yes then the team has home-court

  8. Number of Boards: provide the number of players that is team is comprised of, i.e. 8 total players

There are two butons on this panel:

  1. create: when clicked it will create a team frame with the current provided information on the above fields.

  2. clear: when clicked it will empty the entries to all of the above fields.

Notes:

  • Teams are identified by there NAMES. You CANNOT create two teams with the same name.

  • If a team's frame is already created but the provided informations need to be adjusted simple provide the team's name in the appropriate field and all adjusted information and the click apply. No new frame will be created but the informations on the existing one will be updated.

3. Players Database

On this panel the information about the club's players are displayed. On the top there are a few general information about the Club's Name, total number of players and last time the players database was updated. The players informations are organized in a table-dataframe manner as rows and columns. The table is interactive in the following operations:

  • when click on the name of a column the rows will be sorted in ascended order of the column values. When clicked on the same name order is descended.

  • when double click on a players fide id value, the players fide profile will be opened in the browser

  • you can select multiple players by pressing crt + left click and place those on a team via the middle-right panel.

4. Created Teams Frame

On this panel the frames of the created teams are displayed with the provided information and with the spedified number of players to be filled. There are 3 sections on this panel:

  • on the top the team's information is displayed, with an additional field for any optional additional information to be entered, i.e. some useful telephone numbers for upcoming match.

  • in the middle there is a similar database table with the players of this team. Multiple players can be selected.

  • in the bottom there are 5 buttons:

    1. Remove team: remove the created frame of the team

    2. Remove Selected Players: removes the currently selected players out of the team's composition.

    3. Move Player Up/Down: when pressed move the currently selected player on place up/down in the team composition order.

    4. Export to pdf: when pressed creates a pdf with the current state of the team in the same folder with name the name of the team.

Notes:

  • You can switch between the teams frames

  • You can fill a team with players using the middle-right panel appropriately

  • You cannot place the same player twice on the same team

  • You could place the same player twice on different teams, however there is a warning for that to avoid confusions

  • Any restrictions on the teams composition is to be handled manualy by the user. For example the order of the players, elo and age restrictions per board number, etc.

  • When export to a pdf, the columns currently present on the table will be present. If some additional columns are needed, impose those on the team's table by using the top-right panel appropriately.

5. Configure Table Columns

Use this panel to add/remove and configure the order of the players columns in both the players database panel and on teams frame separetely. The columns and order to be applyed to either the players Database or to a team frame is presented on the list box and configured appropriately by the operation of this panel.

There are 8 widgets on this panel:

  1. Columns to show: when clicked the list of all avaliable columns is displayed. Select the column you want to add or remove.

  2. Add/Remove Buttons: when pressed adds/remove the selected column to the list of columns displayed below. The added column is always placed last

  3. Move up/down Buttons: when a column name is selected from the list of columns it moves that column up/down the list to the desired order.

  4. Reset Button: reset the columns list to the default list

  5. Clear Button: empty the columns list

  6. Columns list: the list of columns and order to be applyed to the database or a team's frame

  7. Teams list: the list of teams created and the overall database. Select the one you want to configure its columns. When a team is created is added to this list.

  8. Apply Columns Button: apply the current list and order of columns to the selected team or to the database.

6. Fill Teams with Selected Players

Use this panel to place players from the database to a team's composition with the following two widges:

  1. Select Team list: select a team from the created teams to place players to.

  2. Move Players to Team Button: When pressed the currently selected players from the database will be place to the selected team.

Notes:

  • Select one or multiple players from the database using ctrl + left click, then select the desired team to place those players and hit the Move Players to Team button.

  • A player cannot be twice on the same team, but it CAN be on two different teams

  • the total number of players cannot excide the number of players of the team

  • the players order on the team is to be handled by the user manually and appropriately each time depending on the restrictions of the tournament, i.e. elo limits, age restrictions, etc.

7. Update Database

On this panel you can create a club's players database combining both national and internation information about the players. using the following:

  • Enter Club's URL: enter the ESO club's URL, i.e https://chesstu.be/eso/club/01151

  • Create/Update Database: when pressed will create/update the database of the provided url's club

  • Time-elapsed bar

Notes:

  • Database: the programm will create a database with the players information from BOTH the eso and fide sites as a .pkl file on the same folder location as the program's executable file and a .json file with metadata of the Club.

  • If the Club's database has already been created its URL is listed on the box and pressing the Update button will update the current database to the latest player's information.

  • At the begining of each month a reminder window is displayed to the user to update the database to fetch the latest elo.

  • If multiply clubs database exist on the folder the program will automatially display the latest updated one
